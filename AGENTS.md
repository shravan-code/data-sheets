# AGENTS.md

This project is a data processing repository containing code for working with various data technologies.

## Project Structure

- `src/ci-cd/` - CI/CD pipelines
- `src/pandas/` - Pandas data processing scripts
- `src/postgresql/` - PostgreSQL utilities and scripts
- `src/python/` - General Python utilities
- `src/spark/` - Apache Spark processing scripts

## Tech Stack

- Python
- Pandas
- Apache Spark
- PostgreSQL
- CI/CD (GitHub Actions or similar)

## Commands

- Python tests: `pytest` or `python -m pytest`
- Linting: `ruff check .` or `flake8`
- Type checking: `mypy`

## Conventions

- Follow PEP 8 style guide for Python code
- Use type hints where appropriate
- Write docstrings for public functions and classes
