Python Methods

Built-in functions and methods for strings, lists, dicts, and sets.

Built-in Functions

FunctionDescriptionExample
len(x)Lengthlen("hi") → 2
range(n)Sequencerange(5) → 0..4
enumerate(x)Index + valueenumerate(['a','b'])
zip(a, b)Combinezip([1,2],[3,4])
map(f, x)Apply functionmap(str.upper, ['a'])
filter(f, x)Keep Truefilter(None, [0,1])
sorted(x)Sortsorted([3,1])
sum(x)Sumsum([1,2]) → 3
min/max(x)Min/Maxmin([1,2]) → 1

String Methods

MethodDescription
s.upper()Uppercase
s.lower()Lowercase
s.split(sep)Split into list
s.join(list)Join with separator
s.strip()Remove whitespace
s.replace(a, b)Replace text
s.find(x)Index of x (-1 if not found)
s.startswith(x)Starts with x
s.isdigit()All digits
"hello".upper()           # "HELLO"
",".join(['a','b'])      # "a,b"
"  hi  ".strip()         # "hi"

List Methods

MethodDescription
lst.append(x)Add to end
lst.extend(x)Add all items
lst.insert(i, x)Insert at index
lst.pop()Remove last
lst.remove(x)Remove first x
lst.sort()Sort in-place
lst.reverse()Reverse
lst.index(x)Index of x
append vs extend: [1].append([2]) → [1, [2]]
[1].extend([2]) → [1, 2]

Dict Methods

MethodDescription
d.get(k, default)Get value or default
d.keys()All keys
d.values()All values
d.items()Key-value pairs
d.pop(k)Remove and return
d.update(other)Merge dict
d = {'a': 1, 'b': 2}
d.get('c', 0)            # 0 (default)
d | {'c': 3}            # {'a':1, 'b':2, 'c':3} (Python 3.9+)

Set Methods

OperationMethod
Uniona | b
Intersectiona & b
Differencea - b
Symmetric Diffa ^ b

{1,2} & {2,3}   # {2}
