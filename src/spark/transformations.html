PySpark Transformations

Narrow Transformation: Data stays in the same partition. No data moves between machines. Fast & cheap.
Wide (Shuffle) Transformation: Data moves between partitions/machines over the network. Slower but necessary.

Selection & Columns (Narrow)

MethodDescription
.select()Pick specific columns
.selectExpr()Select using SQL expressions
.withColumn()Add or replace a column
.withColumnRenamed()Rename a column
.drop()Remove columns
.cast()Change data type

Filtering (Narrow)

MethodDescription
.filter()Keep rows matching condition
.where()Alias for filter()
.limit()Return first N rows
.sample()Random sample of rows

Sorting

MethodDescription
.orderBy() WIDESort globally (shuffles data)
.sortWithinPartitions()Sort within each partition (no shuffle)

Deduplication

MethodDescription
.distinct() WIDERemove completely duplicate rows
.dropDuplicates() WIDERemove duplicates based on specific columns

Aggregations (Wide)

# Basic groupBy + agg
df.groupBy('dept').agg(
    count('*').alias('count'),
    sum('salary').alias('total'),
    avg('salary').alias('avg')
).show()

# Multiple aggregations
df.groupBy('dept').agg(
    count('*').alias('headcount'),
    avg('salary').alias('avg_salary')
).show()

# PIVOT
df.groupBy('dept').pivot('city').count().show()

Joins (Wide)

TypeResult
"inner"Only matching rows from both sides
"left"All left rows + matching right (null if no match)
"right"All right rows + matching left
"outer"All rows from both sides
"semi"Left rows where a match exists in right
"anti"Left rows where NO match exists in right
.crossJoin()Cartesian product (every x every)
broadcast()No shuffle - copy small table to all nodes

Reshaping

# Union (stack rows)
df1.union(df2).show()

# Explode array
df.select('name', F.explode('skills').alias('skill')).show()

# Split string to array
df.withColumn('parts', F.split(col('city'), ' ')).show()

# Array from columns
df.withColumn('profile', F.array('name', 'dept', 'city')).show()

# Struct (nested)
df.withColumn('info', F.struct('name', 'age', 'salary')).show()

String Operations (Narrow)

# Concatenate
df.withColumn('combined', F.concat_ws('_', 'dept', 'city')).show()

# Replace
df.withColumn('clean', F.regexp_replace('city', ' ', '_')).show()

# Conditional strings
df.withColumn('band',
    F.when(col('salary') > 90000, 'High')
     .when(col('salary') > 70000, 'Medium')
     .otherwise('Low')
).show()

Window Functions

from pyspark.sql.window import Window
from pyspark.sql.functions import row_number, rank, dense_rank, lag, lead

# Define window
w = Window.partitionBy('dept').orderBy(F.desc('salary'))

# Ranking
df.withColumn('rank', row_number().over(w)).show()
df.withColumn('rank', rank().over(w)).show()
df.withColumn('dense_rank', dense_rank().over(w)).show()

# Lag/Lead
w2 = Window.partitionBy('dept').orderBy('join_year')
df.withColumn('prev_salary', lag('salary', 1).over(w2)).show()
df.withColumn('next_salary', lead('salary', 1).over(w2)).show()

# Cumulative sum
w3 = Window.orderBy('salary').rowsBetween(Window.unboundedPreceding, Window.currentRow)
df.withColumn('cumsum', F.sum('salary').over(w3)).show()

# NTile (buckets)
df.withColumn('quartile', F.ntile(4).over(w)).show()

Partitioning

# Check partitions
df.rdd.getNumPartitions()

# Repartition (full shuffle, increases or decreases)
df.repartition(20).show()
df.repartition(10, 'dept').show()  # hash partition by column

# Coalesce (reduces, NO shuffle - faster!)
df.coalesce(4).show()

# Repartition by range
df.repartitionByRange(4, 'salary').show()

Array & Map Functions

from pyspark.sql.functions import (
    array, array_contains, array_distinct,
    array_max, array_min, array_sort,
    explode, flatten, size

# Create array
df.withColumn('arr', F.array('col1', 'col2')).show()

# Array operations
df.select(

    array_contains('skills', 'Python').alias('has_python'),

UDFs

from pyspark.sql.functions import udf
from pyspark.sql.types import StringType

# Regular UDF (slower - Python per row)
@udf(returnType=StringType())
def classify_salary(salary):
    if salary >= 100000: return 'Senior'
    if salary >= 80000: return 'Mid'
    return 'Junior'

df.withColumn('level', classify_salary(col('salary'))).show()

# Pandas UDF (faster - vectorized)
import pandas as pd
from pyspark.sql.functions import pandas_udf

@pandas_udf(DoubleType())
def salary_with_tax(salary: pd.Series) -> pd.Series:
    return salary * 0.7

df.withColumn('net_salary', salary_with_tax(col('salary'))).show()
