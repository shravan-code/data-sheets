# **1\. Setup & Connection**

| Scenario | 🐘 PostgreSQL | 🐼 Pandas | 🔢 NumPy | ⚡ PySpark |
| :---- | :---- | :---- | :---- | :---- |
| **Install package** | `pip install psycopg2-binary # or psycopg2` | `pip install pandas` | `pip install numpy` | `pip install pyspark` |
| **Import library** | `import psycopg2` | `import pandas as pd` | `import numpy as np` | `from pyspark.sql import SparkSession from pyspark.sql.functions import * from pyspark.sql.window import Window` |
| **Create a connection/session** | `conn = psycopg2.connect(   host='localhost',   dbname='mydb',   user='user',   password='pass' ) cur = conn.cursor()` | `# No persistent connection # Works on in-memory DataFrames # For DB: pd.read_sql(sql, conn)` | `# No connection needed # Operates on in-memory arrays` | `spark = SparkSession.builder \   .appName('MyApp') \   .master('local[*]') \   .getOrCreate()` |
| **Load data (CSV)** | `COPY tbl FROM '/path/f.csv'   WITH (FORMAT CSV,HEADER); # OR via psycopg2: cur.execute("COPY tbl FROM...")` | `df = pd.read_csv('file.csv') # with options: df = pd.read_csv('f.csv',   sep=',', header=0,   dtype={'col':str},   parse_dates=['dt'])` | `arr = np.loadtxt('f.csv',   delimiter=',', skiprows=1) # structured: dt = np.dtype([('a','f8'),('b','U10')]) arr = np.genfromtxt('f.csv',   delimiter=',',names=True)` | `df = spark.read.csv('path',   header=True,   inferSchema=True) # or with schema: df = spark.read.option(   'header','true').csv('path')` |

| **Close / stop** | `cur.close() conn.close()` | `# No explicit close # GC handles memory` | `# No explicit close` | `spark.stop()` |

# **3\. Filtering & Selection**

| Scenario | 🐘 PostgreSQL | 🐼 Pandas | 🔢 NumPy | ⚡ PySpark |
| :---- | :---- | :---- | :---- | :---- |
| **Simple WHERE / filter** | `SELECT *  FROM tbl  WHERE age > 30;` | `df[df['age'] > 30] df.query('age > 30')` | `arr[arr > 30] arr[arr[:, 0] > 30]  # by col idx` | `df.filter(df.age > 30) df.where(col('age') > 30)` |
| **AND / OR conditions** | `SELECT *  FROM tbl WHERE age > 30   AND city = 'Delhi';` | `df[(df.age > 30) & (df.city=='Delhi')] df.query("age>30 and city=='Delhi'")` | `mask=(arr[:,0] > 30) & (arr[:,1] == 'Delhi') arr[mask]` | `df.filter( (df.age > 30) &      (df.city == 'Delhi'))` |
| **NOT / \!= condition** | `SELECT *  FROM tbl WHERE status <> 'inactive';` | `df[df.status != 'inactive'] df[~(df.status == 'inactive')]` | `arr[arr != 0] arr[~np.isin(arr, [0])]` | `df.filter( df.status !='inactive') df.filter( ~(col('status') == 'inactive'))` |
| **IN list** | `SELECT *  FROM tbl WHERE city IN ('Delhi','Mumbai');` | `df[df.city.isin( ['Delhi', 'Mumbai']) ]` | `np.isin(arr, [1, 2, 3]) arr[np.isin(arr,[1,2,3])]` | `df.filter( col('city').isin(   ['Delhi','Mumbai']))` |
|  **NOT IN** |  `SELECT *  FROM tbl WHERE city NOT IN ('Delhi','Mumbai');` |  `df[~df.city.isin( ['Delhi',' Mumbai'])]` |  `arr[~np.isin(arr, [1,2,3])]` |  `df.filter(~col('city').isin(   ['Delhi','Mumbai']))` |
| **BETWEEN range** | `SELECT *  FROM tbl WHERE age BETWEEN 20 AND 40;` | `df[ df.age.between(20, 40)] df[ (df.age>=20) & (df.age<=40)]` | `arr[(arr>=20) & (arr<=40)]` | `df.filter( col('age').between(20,40))` |
| **IS NULL / IS NOT NULL NULL → missing value NaN → invalid numeric value**  | `SELECT *  FROM tbl  WHERE col IS NULL; SELECT *  FROM tbl  WHERE col IS NOT NULL; COALESCE(col, 0)` | `df[df.col.isna()] df[df.col.notna()]` | `arr[np.isnan(arr)] arr[~np.isnan(arr)]` | `df.filter( col('c').isNull()) df.filter( col('c').isNotNull()) df.filter( isnan("col")).show()`  |
| **LIKE / string contains** | `SELECT *  FROM tbl WHERE name LIKE '%kumar%';` | `df[df.name.str.contains('kumar')] df[df.name.str.startswith('A')] df[df.name.str.endswith('a')]` | `mask=np.char.find( arr.astype(str),'kumar') >= 0 arr[mask]` | `df.filter(col('name').like('%kumar%')) df.filter(col('name').contains('kumar')) df.filter(col('name').startswith('A'))` |
| **Select specific columns** | `SELECT name, age, city FROM tbl;` | `Df[['name','age','city']] df.loc[:, ['name','age','city']]` | `arr[:, [0, 2, 4]]  # by index` | `df.select('name','age','city') df.select(col('name'),col('age'))` |
| **Limit / TOP N rows** | `SELECT * FROM tbl  LIMIT 10; SELECT * FROM tbl  FETCH FIRST 10 ROWS ONLY;` | `df.head(10) df.iloc[:10]` | `arr[:10] arr[:10, :]` | `df.limit(10)` |

# **4\. Sorting**

| Scenario | 🐘 PostgreSQL | 🐼 Pandas | 🔢 NumPy | ⚡ PySpark |
| :---- | :---- | :---- | :---- | :---- |
| **Sort ascending** | `SELECT *  FROM tbl  ORDER BY salary ASC;` | `df.sort_values('salary') df.sort_values('salary', ascending=True)` | `arr[ arr[:, 1].argsort()]` | `df.sort('salary') df.orderBy('salary') df.orderBy( col('salary').asc())` |
| **Sort descending** | `SELECT *  FROM tbl  ORDER BY salary DESC;` | `df.sort_values('salary', ascending=False)` | `arr[ arr[:,1].argsort() [::-1] ]` | `df.sort( col('salary').desc()) df.orderBy( col('salary').desc())` |
| **Sort by multiple columns** | `SELECT *  FROM tbl ORDER BY dept ASC, salary DESC;` | `df.sort_values( ['dept','salary'], ascending=[True,False])` | `idx = np.lexsort(   (-arr[:,1],    arr[:,0]))  arr[idx]` | `df.orderBy( col('dept').asc(),      col('salary').desc())` |
| **Sort strings ( nulls last)** | `SELECT *  FROM tbl ORDER BY name ASC  NULLS LAST;` | `df.sort_values('name',   na_position='last')` | `# np.char sorts lexicographically np.sort(arr.astype(str))` | `df.orderBy(col('name').asc_nulls_last()) df.orderBy(col('name').desc_nulls_first())` |
| **Get top N per group (rank)** | `SELECT *  FROM (   SELECT *,   ROW_NUMBER()    OVER (     PARTITION BY dept     ORDER BY salary   DESC) as rn   FROM tbl) tbl_x  WHERE rn <= 3;` | `df['rn'] = df.groupby('dept') ['salary'].rank(   method='first',   ascending = False) df[df.rn <= 3]` | `# sort within groups manually # then slice first 3 per group` | `w = Window.partitionBy('dept') \    .orderBy(col('salary').desc()) df.withColumn('rn', row_number().over(w)) df = df.filter(col('rn') <= 3)` |

# **5\. Aggregation & Group By**

| Scenario | 🐘 PostgreSQL | 🐼 Pandas | 🔢 NumPy | ⚡ PySpark |
| :---- | :---- | :---- | :---- | :---- |
| **COUNT per group** | `SELECT dept, COUNT(*) FROM tbl  GROUP BY dept;` | `df.groupby('dept').size().reset_index(   name='count') df.groupby('dept')['id'].count()` | `vals, cnts = np.unique(   dept_arr, return_counts=True)` | `df.groupBy('dept'). count().show()` |
| **SUM per group** | `SELECT dept,SUM(salary) FROM tbl  GROUP BY dept;` | `df.groupby('dept') ['salary'].sum()` | `np.add.reduceat(   arr_sorted[:,1], group_idx)` | `df.groupBy('dept').sum('salary') df.groupBy('dept'). agg(sum('salary').alias('total'))` |
|  **AVG / MEAN per group** |  `SELECT dept, AVG(salary) FROM tbl  GROUP BY dept;` |  `df.groupby('dept') ['salary'].mean()` |  `# compute per-group manually means = {} for g in np.unique(groups):   means[g]=np.mean(arr[groups==g])` |  `df.groupBy('dept'). avg('salary') df.groupBy('dept'). agg(avg('salary').alias('avg_sal'))` |
| **MIN & MAX per group** | `SELECT dept, MIN(salary),  MAX(salary) FROM tbl  GROUP BY dept;` | `df.groupby('dept') ['salary'].agg(       ['min','max'])` | `np.min(arr, axis=0) np.max(arr, axis=0)` | `df.groupBy('dept'). agg( min('salary'). alias('min_s'), max('salary'). alias('max_s'))` |
| **Multiple aggregations** | `SELECT dept,   COUNT(*),    AVG(sal),   SUM(sal),    MIN(sal),    MAX(sal) FROM tbl  GROUP BY dept;` | `df.groupby('dept'). agg(   cnt=('sal','count'),   avg_s=('sal','mean'),   sum_s=('sal','sum') )` | `# axis=0 for col-wise stats np.apply_along_axis(   lambda x:[x.mean(),x.sum()],   0, arr)` | `df.groupBy('dept').agg( count('*'). alias('cnt'), avg('sal'). alias('avg'), sum('sal'). alias('total'), min('sal'). alias('min_s'))` |
| **HAVING ( filter on aggregate)** | `SELECT dept, AVG(salary) avg_s FROM tbl  GROUP BY dept HAVING AVG(salary) > 70000;` | `df.groupby('dept') ['salary']   .mean().reset_index()   .query( 'salary > 70000') )` | `means = np.array([...])  means[means > 70000]` | `df.groupBy('dept') .agg( avg('salary'). alias('a')). filter( col('a') > 70000 )` |
| **Count distinct values** | `SELECT  COUNT(DISTINCT city) FROM tbl;` | `df['city'].nunique()` | `len(np.unique(arr))` | `df.select( countDistinct('city')).show()` |
| **Running / Cumulative total** | `SELECT  id,  sal, SUM(sal) OVER(ORDER BY id) AS running_total FROM tbl;` | `df['sal'].cumsum() # per group: df.groupby( 'dept')['sal'].cumsum()` | `np.cumsum(arr) np.cumsum(arr, axis=0)` | `w = Window.orderBy('id') df.withColumn('run',   sum('sal').over(w))` |
| **Percentage of total** | `SELECT  sal, sal * 100.0 / SUM(sal)  OVER () AS pct FROM tbl;` | `df.sal / df.sal.sum() * 100` | `pct = arr / arr.sum() * 100` | `w = Window.rowsBetween( Window.unboundedPreceding, Window.unboundedFollowing) df.withColumn('pct', col('sal')/ sum('sal').over(w)*100)` |

# **6\. Joins & Merges**

| Scenario | 🐘 PostgreSQL | 🐼 Pandas | 🔢 NumPy | ⚡ PySpark |
