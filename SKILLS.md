# SKILLS.md

This document defines specialized skills for working with this data processing project.

## Data Processing

When working with data processing tasks:

- Use Pandas for tabular data manipulation
- Use Spark for large-scale distributed processing
- Consider memory efficiency when processing large datasets
- Use vectorized operations over loops where possible

## Python Development

- Follow PEP 8 style guidelines
- Use type hints for function signatures
- Write descriptive docstrings
- Handle exceptions appropriately
- Log important operations

## Database (PostgreSQL)

- Use parameterized queries to prevent SQL injection
- Close database connections properly (use context managers)
- Handle connection pooling for production workloads

## CI/CD

- Ensure tests pass before merging
- Include linting and type checking in pipelines
- Use environment variables for configuration
