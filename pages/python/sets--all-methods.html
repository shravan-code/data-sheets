Section 3: Foundations

Sets — All Methods

Sets are unordered collections of unique hashable elements. O(1) average for add/remove/lookup.

Basic Operations: add, remove, discard, pop

Modify set elements. remove() raises KeyError if the item is missing; discard() does not. pop() removes an arbitrary element.

s = {1, 2, 3}
s.add(4)
s.remove(1)
s.discard(10)  # no error
print(s.pop()) # returns arbitrary
print(s)
Output
2
{3, 4}

Set Mathematics: union, intersection, difference

Standard set theory operations. Can be performed using methods or operators (| for union, & for intersection, - for difference).

a = {1, 2, 3}
b = {3, 4, 5}
print(a | b)   # union: {1, 2, 3, 4, 5}
print(a & b)   # intersection: {3}
print(a - b)   # difference: {1, 2}
Output
{1, 2, 3, 4, 5}
{3}
{1, 2}

Membership & Subsets

Check relationships between sets. Efficient O(1) average time complexity.

a = {1, 2}
b = {1, 2, 3}
print(a.issubset(b))   # True
print(b.issuperset(a)) # True
print(a.isdisjoint({4, 5})) # True

frozenset

Immutable set — hashable, usable as dict key.

fs = frozenset([1, 2, 3])
print(fs)

# Can use as dict key
graph = {frozenset({'A','B'}): 5}
print(graph)
Output
frozenset({1, 2, 3})
{frozenset({'A', 'B'}): 5}

Interview Mastery

Q: How would you remove duplicates from a list while preserving order?

A: Python 3.7+: `list(dict.fromkeys(lst))` — dicts preserve insertion order. Or: `seen = set(); result = [x for x in lst if not (x in seen or seen.add(x))]`. Simple `list(set(lst))` removes duplicates but loses order.
