Section 1: Foundations

Operators

Python supports arithmetic, comparison, logical, bitwise, assignment, identity, and membership operators.

Arithmetic: Basic Math

Standard operations like addition, subtraction, multiplication, and division. Division (/) always returns a float.

print(10 + 3)   # 13
print(10 - 3)   # 7
print(10 * 3)   # 30
print(10 / 3)   # 3.333...
Output
13
7
30
3.3333333333333335

Arithmetic: Advanced Math

Floor division (//) truncates results; modulo (%) returns the remainder; exponentiation (**) raises to power.

print(10 // 3)  # 3
print(10 % 3)   # 1
print(2 ** 3)   # 8

Comparison Operators

Return boolean values. Python supports chained comparisons.

print(5 == 5)    # True
print(5 != 3)    # True
print(5 > 3)     # True
print(5 < 3)     # False
print(5 >= 5)    # True
print(5 <= 4)    # False

# Chained comparison (Pythonic!)
x = 5
print(1 < x < 10)   # True
print(1 < x < 4)    # False

Logical Operators

and, or, not with short-circuit evaluation. Return the actual values, not just True/False.

# Short-circuit: returns first falsy for 'and', first truthy for 'or'
print(True and False)   # False
print(True or False)    # True
print(not True)         # False

# Returns actual values (useful pattern)
print(0 or 'default')   # 'default' (0 is falsy)
print(5 or 'default')   # 5 (5 is truthy)
print(None and 'never') # None (short-circuits)
print([] or {} or 42)   # 42 (first truthy)
Output
False
True
False
default
5
None
42

Bitwise Operators

Operate on integer bits directly.

a, b = 0b1010, 0b1100  # 10 and 12
print(bin(a & b))   # AND:  0b1000 = 8
print(bin(a | b))   # OR:   0b1110 = 14
print(bin(a ^ b))   # XOR:  0b0110 = 6
print(bin(~a))      # NOT:  -11 (two's complement)
print(bin(a << 1))  # Left shift:  0b10100 = 20
print(bin(a >> 1))  # Right shift: 0b0101 = 5
Output
0b1000
0b1110
0b110
-0b1011
0b10100
0b101

Identity & Membership

is/is not for object identity; in/not in for membership testing.

# Identity
a = [1, 2]
b = a
c = [1, 2]
print(a is b)    # True - same object
print(a is c)    # False - different objects
print(a == c)    # True - equal values

# Membership
print(3 in [1, 2, 3])        # True
print('py' in 'python')      # True
print('key' in {'key': 1})   # True (checks keys)
print(5 not in range(3))     # True
Output
True
False
True
True
True
True
True

Interview Mastery

Q: What is operator precedence in Python?

A: From highest to lowest: ** (exponent), ~+- (unary), */%// , +-, <<>>, &, ^, |, comparisons, not, and, or. Use parentheses to make intent explicit.

Q: How does Python's `or` operator work for default values?

A: `x = value or default` returns `value` if truthy, else `default`. But beware: if value is 0, '', [], etc. (valid but falsy), it returns default unexpectedly. Prefer `x = value if value is not None else default` for precision.
