Section 3: Foundations

Lists — All Methods

Lists are ordered, mutable sequences. They can contain mixed types.

Adding Elements: append, extend, insert

Add single items, merge iterables, or inject at specific positions. append() is O(1), while insert() is O(n).

lst = [1, 2]
lst.append(3)         # [1, 2, 3]
lst.extend([4, 5])    # [1, 2, 3, 4, 5]
lst.insert(0, 0)      # [0, 1, 2, 3, 4, 5]
print(lst)

Removing Elements: pop, remove, clear

Remove items by index (pop), value (remove), or wipe the entire list. pop() from the end is O(1); remove() is O(n).

lst = [10, 20, 30, 40]
print(lst.pop())      # 40 (removes last)
print(lst.pop(0))     # 10 (removes index 0)
lst.remove(30)        # removes value 30
print(lst)            # [20]
lst.clear()           # empty the list
print(lst)
Output
40
10
[20]
[]

Searching: index & count

Find the position of an item or count its occurrences. Both operations are O(n).

lst = ['a', 'b', 'a', 'c']
print(lst.index('b')) # 1
print(lst.count('a')) # 2

Ordering: sort & reverse

sort() modifies the list in-place (Timsort, O(n log n)). reverse() flips the order in-place (O(n)).

lst = [3, 1, 4, 2]
lst.sort()            # [1, 2, 3, 4]
print(lst)
lst.reverse()         # [4, 3, 2, 1]
print(lst)

sorted() vs sort()

sorted() returns new list; sort() modifies in-place.

nums = [3, 1, 4, 1, 5]
result = sorted(nums)          # new list
print(nums)                    # original unchanged
print(result)

# Sort strings by length
words = ['banana', 'apple', 'fig', 'cherry']
print(sorted(words, key=len))

# Sort by multiple criteria
people = [('Alice', 30), ('Bob', 25), ('Charlie', 30)]
print(sorted(people, key=lambda x: (x[1], x[0])))
Output
[3, 1, 4, 1, 5]
[1, 1, 3, 4, 5]
['fig', 'apple', 'banana', 'cherry']
[('Bob', 25), ('Alice', 30), ('Charlie', 30)]

Shallow vs Deep Copy

Shallow copy copies the list but not nested objects. Deep copy copies everything.

import copy

# Shallow copy - nested objects still shared
original = [[1, 2], [3, 4]]
shallow = original.copy()  # or list(original) or original[:]
shallow[0].append(99)
print(original)  # [[1,2,99],[3,4]] - inner list affected!

# Deep copy - fully independent
original2 = [[1, 2], [3, 4]]
deep = copy.deepcopy(original2)
deep[0].append(99)
print(original2)  # [[1,2],[3,4]] - unchanged
Output
[[1, 2, 99], [3, 4]]
[[1, 2], [3, 4]]

Interview Mastery

Q: What is the time complexity of list operations?

A: append: O(1) amortized, pop from end: O(1), pop from index: O(n), insert at index: O(n), index lookup by index: O(1), search by value: O(n), sort: O(n log n).

Q: How is list.sort() different from sorted()?

A: list.sort() modifies the list in-place and returns None. sorted() returns a new list and works on any iterable. Both accept key and reverse parameters. list.sort() is slightly faster since it doesn't create a new list.
