Slowly Changing Dimensions (SCD Types)

Available Types

type name description
SCD Type 0 Retain Original No changes allowed. Keep original value forever.
SCD Type 1 Overwrite Overwrite old value. No history kept.
SCD Type 2 Add New Row Expire old row, insert new row with dates. Full history kept.
SCD Type 3 Add New Column Keep current and previous value in separate columns. Limited history.

General syntax for Slowly Changing Dimensions (SCD Types)

-- Pattern for Slowly Changing Dimensions (SCD Types)
SELECT column_a, column_b 
FROM source_table 
WHERE logic_condition = TRUE;
Output
Result set returned based on the specified logic.
