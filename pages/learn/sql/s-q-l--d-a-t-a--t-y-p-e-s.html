Section 1: Fundamentals

SQL Data Types

SQL data types define what kind of value a column can hold.

Numeric

type description example
INT / INTEGER Whole numbers 42
BIGINT Large whole numbers 9876543210
DECIMAL(p,s) Fixed precision decimals 99.95
NUMERIC(p,s) Exact numeric 123.456
FLOAT/REAL Approximate floating point 3.14

String

type description example
CHAR(n) Fixed-length string 'ABC'
VARCHAR(n) Variable-length string 'Hello'
TEXT Unlimited-length string 'Long text...'

Date/Time

type description example
DATE Date only '2024-01-15'
TIME Time only '14:30:00'
TIMESTAMP Date and time '2024-01-15 14:30:00'
INTERVAL Duration '2 days'

Boolean

type description example
BOOLEAN True/False TRUE

JSON/Array (PostgreSQL)

type description example
JSON JSON text '{"key": "val"}'
JSONB Binary JSON (indexable) '{"id": 1}'
ARRAY Array of any type ARRAY[1,2,3]

General syntax for SQL Data Types

-- Pattern for SQL Data Types
SELECT column_a, column_b 
FROM source_table 
WHERE logic_condition = TRUE;
Output
Result set returned based on the specified logic.
Pro Tips & Tricks

  • Use EXPLAIN to see how the database handles your query
