18. Output & Formatting

Controlling how results are displayed and exported.

Formatting Cmdlets


Format-Table, Format-List, Format-Wide, and Out-* cmdlets.


Output Formatting


# Format as table
Get-Process | Format-Table Name, Id, CPU, WorkingSet -AutoSize
Get-Process | Format-Table Name, @{N='MemMB';E={[math]::Round($_.WS/1MB,1)}} -AutoSize

# Format as list
Get-Process | Select-Object -First 2 | Format-List *

# Format wide
Get-Process | Format-Wide -Property Name -Column 4

# Output to various destinations
Get-Process | Out-GridView                   # Interactive GUI grid
Get-Process | Out-File 'procs.txt'
Get-Process | Out-String
Get-Process | Out-Null                       # Discard output
Get-Process | ConvertTo-Html | Out-File 'procs.html'
Get-Process | ConvertTo-Csv
Get-Process | ConvertTo-Json

# Write output methods
Write-Host 'Console only'              # Only to host, not pipeline
Write-Output 'To pipeline'            # Goes to pipeline
Write-Verbose 'Debug when -Verbose'   # Only with -Verbose
Write-Warning 'A warning message'     # Yellow warning
Write-Error 'An error message'        # Error stream
Write-Debug 'Debug message'           # Only with -Debug
Write-Progress -Activity 'Processing' -PercentComplete 50
