5. Control Flow

Conditional statements and loops to control program execution.

If / ElseIf / Else


Standard conditional branching in PowerShell.


If Statements


$score = 85

if ($score -ge 90) {
    Write-Host 'Grade: A'
} elseif ($score -ge 80) {
    Write-Host 'Grade: B'
} elseif ($score -ge 70) {
    Write-Host 'Grade: C'
} else {
    Write-Host 'Grade: F'
}

# One-liner
if ($score -ge 60) { Write-Host 'Passed' } else { Write-Host 'Failed' }


Switch Statement


Switch is more powerful than in many languages — it supports wildcards, regex, and multiple conditions.


Switch Examples


$day = 'Monday'

switch ($day) {
    'Monday'   { Write-Host 'Start of work week' }
    'Friday'   { Write-Host 'End of work week' }
    'Saturday' { Write-Host 'Weekend!' }
    'Sunday'   { Write-Host 'Weekend!' }
    default    { Write-Host 'Midweek' }
}

# Multiple matches allowed
$num = 5
switch ($num) {
    { $_ -lt 10 } { Write-Host 'Less than 10' }
    { $_ -gt 3  } { Write-Host 'Greater than 3' }
}

# Wildcard switch
switch -Wildcard ('PowerShell') {
    'Power*' { Write-Host 'Starts with Power' }
    '*Shell' { Write-Host 'Ends with Shell' }
}

# Regex switch
switch -Regex ('user123@email.com') {
    '^\w+@'    { Write-Host 'Looks like email' }
    '\d+$'     { Write-Host 'Ends with numbers (not matched here)' }
}

# Switch over array
switch (1,2,3,4,5) {
    { $_ % 2 -eq 0 } { Write-Host "$_ is Even" }
    { $_ % 2 -ne 0 } { Write-Host "$_ is Odd" }
}


Loops


PowerShell offers for, foreach, while, do-while, and do-until loops.


For Loop


for ($i = 0; $i -lt 5; $i++) {
    Write-Host "Iteration: $i"
}

# Countdown
for ($i = 10; $i -ge 0; $i--) {
    Write-Host $i
    Start-Sleep -Milliseconds 100
}

Foreach Loop


$fruits = @('Apple', 'Banana', 'Cherry')

foreach ($fruit in $fruits) {
    Write-Host "Fruit: $fruit"
}

# ForEach-Object in pipeline
1..5 | ForEach-Object { $_ * $_ }   # 1 4 9 16 25

# Foreach with index workaround
for ($i = 0; $i -lt $fruits.Count; $i++) {
    Write-Host "[$i] $($fruits[$i])"
}

While & Do Loops


# While loop
$count = 0
while ($count -lt 5) {
    Write-Host "Count: $count"
    $count++
}

# Do-While (executes at least once)
$num = 0
do {
    Write-Host "Num: $num"
    $num++
} while ($num -lt 3)

# Do-Until
$x = 10
do {
    Write-Host $x
    $x--
} until ($x -eq 0)

Loop Control: Break & Continue


# Break exits the loop
for ($i = 0; $i -lt 10; $i++) {
    if ($i -eq 5) { break }
    Write-Host $i
}

# Continue skips to next iteration
for ($i = 0; $i -lt 10; $i++) {
    if ($i % 2 -eq 0) { continue }
    Write-Host "Odd: $i"
}

# Labeled break for nested loops
:outer for ($i = 0; $i -lt 3; $i++) {
    for ($j = 0; $j -lt 3; $j++) {
        if ($j -eq 1) { break outer }
        Write-Host "$i,$j"
    }
}
