7. Pipeline & Object Processing

The pipeline is the most powerful feature of PowerShell, passing objects (not text) between commands.

Pipeline Basics

The | operator passes output from one command as input to the next. Objects retain all their properties.

Basic Pipeline


# Get processes and filter/sort
Get-Process | Where-Object { $_.CPU -gt 10 } | Sort-Object CPU -Descending | Select-Object Name, CPU, ID

# Get files, filter by size, sort
Get-ChildItem C:\ -Recurse -File |
    Where-Object Length -gt 1MB |
    Sort-Object Length -Descending |
    Select-Object Name, @{N='SizeMB';E={[math]::Round($_.Length/1MB,2)}} |
    Format-Table -AutoSize

Where-Object

Filter pipeline objects based on conditions.

Filtering with Where-Object


# Traditional syntax
Get-Process | Where-Object { $_.Name -like 'chrome*' }

# Simplified syntax (PowerShell 3+)
Get-Process | Where-Object Name -like 'chrome*'
Get-Service | Where-Object Status -eq 'Running'

# Multiple conditions
Get-Process | Where-Object {
    $_.WorkingSet64 -gt 100MB -and
    $_.CPU -gt 1
}

# Alias: ?
Get-Process | ? { $_.CPU -gt 100 }

Select-Object

Choose specific properties, create calculated properties, or limit output.

Select-Object Usage


# Select specific properties
Get-Process | Select-Object Name, Id, CPU, WorkingSet64

# Calculated property
Get-Process | Select-Object Name, @{
    Name       = 'MemoryMB'
    Expression = { [math]::Round($_.WorkingSet64 / 1MB, 2) }
}

# First/Last N objects
Get-Process | Select-Object -First 5
Get-Process | Select-Object -Last 5
Get-Process | Select-Object -Skip 10

# Unique values
Get-Process | Select-Object -Property Name -Unique

# Expand a property
Get-Process | Select-Object -ExpandProperty Name

Sort-Object & Group-Object

Sorting and grouping pipeline output.

Sorting and Grouping


# Sort ascending/descending
Get-Process | Sort-Object CPU
Get-Process | Sort-Object CPU -Descending

# Sort by multiple properties
Get-ChildItem | Sort-Object Extension, Name

# Group by property
Get-Process | Group-Object -Property PriorityClass

# Group and count
Get-ChildItem -Recurse | Group-Object Extension |
    Sort-Object Count -Descending |
    Select-Object Name, Count

ForEach-Object

Perform an action on each object in the pipeline.

ForEach-Object


# Process each object
Get-Process | ForEach-Object {
    Write-Host "PID: $($_.Id) - $($_.Name)"
}

# Shorthand with %
1..10 | % { $_ * 2 }

# Begin/Process/End blocks
Get-Process | ForEach-Object `
    -Begin   { Write-Host 'Starting...'; $total = 0 } `
    -Process { $total += $_.WorkingSet64 } `
    -End     { Write-Host "Total Memory: $([math]::Round($total/1MB,2)) MB" }

# Parallel execution (PowerShell 7+)
1..10 | ForEach-Object -Parallel {
    Start-Sleep -Seconds 1
    "Done: $_"
} -ThrottleLimit 5

Measure-Object & Tee-Object

Compute statistics and duplicate pipeline output.

Measure and Tee


# Measure stats
Get-Process | Measure-Object CPU -Sum -Average -Maximum -Minimum
Get-Content file.txt | Measure-Object -Line -Word -Character

# Tee-Object: output to variable AND continue pipeline
Get-Process | Tee-Object -Variable procs | Select-Object Name
$procs  # Still contains all process objects
