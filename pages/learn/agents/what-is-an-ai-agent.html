Section 1: Foundations

What is an AI Agent?

Beyond simple LLM completion.

An AI Agent is a system that uses a Large Language Model (LLM) as its central reasoning engine to autonomously achieve goals by interacting with environment through tools and loops.

The Agent Loop

Agents operate in a continuous cycle: Perceive -> Reason -> Act -> Observe. Unlike standard chatbots, agents can decide when to stop or when to retry a failed tool call.

ReAct Pattern

Reasoning + Acting. The agent verbalizes its 'Thought' before performing an 'Action', then processes the 'Observation'. This significantly improves performance on complex reasoning tasks.

Interview Mastery

Q:

What is the difference between a Chain and an Agent?

A:

A Chain is a hardcoded sequence of steps (DAG), whereas an Agent uses the LLM to dynamically decide the next step based on the current state.
