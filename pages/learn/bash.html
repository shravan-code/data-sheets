1. Introduction to Bash

2. Variables

Variables are assigned without spaces around =. Access them with $ prefix. Variable names are case-sensitive.

3. Data Types & Strings

Bash strings can be manipulated using parameter expansion for length, replacement, case conversion, and more.

4. Input & Output

The read command reads input from stdin. It can read into single or multiple variables.

5. Operators & Arithmetic

Bash supports integer arithmetic via (( )) or $(()), and floating-point via bc or awk.

6. Control Flow

Conditional branching using if/elif/else. The condition can be any command—its exit status determines the branch.

7. Functions

Functions group reusable code. They can accept arguments and return values via exit codes or stdout.

8. String Manipulation

These are the three pillars of text processing in Bash. grep searches, sed transforms, awk processes structured text.

9. File & Directory Operations

Bash provides robust file manipulation through built-in commands.

10. Process Management

Processes can run in the background using & and be managed with jobs, fg, bg.

11. Error Handling

Every command returns an exit code: 0 = success, non-zero = failure. $? holds the last exit code.

12. Regular Expressions

Regular expressions are patterns used with grep, sed, and awk for text matching and extraction.

13. Debugging

Bash offers several built-in debugging tools and strategies.

14. Advanced Topics

Capture command output into a variable using $() or backticks.
