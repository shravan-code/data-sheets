Advanced Transformation with Spark SQL

-- Example: Calculating User Lifetime Value (LTV)
SELECT 
    user_id,
    SUM(order_amount) OVER(PARTITION BY user_id) as total_spent,
    COUNT(*) OVER(PARTITION BY user_id) as order_count
FROM silver_orders;

Schema Validation with Pydantic

Validating incoming JSON records before they reach the data lake.

from pydantic import BaseModel, EmailStr

class UserRecord(BaseModel):
    id: int
    name: str
    email: EmailStr
    age: int

# Usage
try:
    record = UserRecord(**incoming_json)
except Exception as e:
    send_to_dlq(incoming_json)

Common Pitfall: Data Skew

If one partition has 90% of your data, the entire cluster will wait for one worker to finish. Use Salting to redistribute data.
