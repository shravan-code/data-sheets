Inheritance

Single Inheritance

Create a hierarchy where a child class inherits all attributes and methods from a parent. Perfect for sharing code between related objects.

class Animal:
    def speak(self): return 'Sound'

class Dog(Animal):
    def speak(self): return 'Woof!'

Multiple Inheritance & Mixins

Python allows a class to inherit from multiple parents. Mixins are small classes that provide specific functionality to be mixed into other classes.

class Flyer:
    def fly(self): return 'Flying'

class Bird(Flyer, Animal):
    pass

super() & MRO

super() calls the next class in the Method Resolution Order (MRO). MRO ensures a consistent search path in complex multiple inheritance scenarios.

class A:
    def say(self): print('A')

class B(A):
    def say(self): 
        super().say()
        print('B')

B().say()
Output
A
B

Interview Mastery

Q:

What is the MRO and how does Python resolve method calls in multiple inheritance?

A:

MRO (Method Resolution Order) determines which class's method is called. Python uses the C3 linearization algorithm: left-to-right depth-first with no duplicates. Check with `ClassName.__mro__`. super() follows the MRO, not necessarily the direct parent.
