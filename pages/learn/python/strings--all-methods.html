Strings — All Methods

Strings in Python are immutable sequences of Unicode characters. They have a rich set of methods for manipulation.

String Creation & Slicing

Strings can be created with single, double, or triple quotes. Slicing uses [start:stop:step].

s = 'Hello, World!'
print(s[0])       # H
print(s[-1])      # !
print(s[0:5])     # Hello
print(s[7:])      # World!
print(s[:5])      # Hello
print(s[::2])     # Hlo ol!
print(s[::-1])    # !dlroW ,olleH (reversed)
Output
H
!
Hello
World!
Hello
Hlo ol!
!dlroW ,olleH

Case Methods

Methods for changing string case.

s = 'hello WORLD'
print(s.upper())        # HELLO WORLD
print(s.lower())        # hello world
print(s.capitalize())   # Hello world (first char upper)
print(s.title())        # Hello World (each word)
print(s.swapcase())     # HELLO world
Output
HELLO WORLD
hello world
Hello world
Hello World
HELLO world

Search Methods

find returns -1 if not found; index raises ValueError.

s = 'hello world hello'
print(s.find('hello'))          # 0 (first occurrence)
print(s.find('hello', 1))       # 12 (search from index 1)
print(s.rfind('hello'))         # 12 (from right)
print(s.find('xyz'))            # -1 (not found)
# s.index('xyz')               # raises ValueError
print(s.count('hello'))         # 2
print(s.startswith('hello'))    # True
print(s.endswith('hello'))      # True
print(s.startswith('world', 6)) # True (from index 6)
Output
0
12
12
-1
2
True
True
True

Check Methods (is*)

Boolean check methods for string content.

print('abc'.isalpha())      # True - all letters
print('123'.isdigit())      # True - all digits
print('abc123'.isalnum())   # True - letters or digits
print('   '.isspace())      # True - all whitespace
print('hello'.islower())    # True
print('HELLO'.isupper())    # True
print('Hello World'.istitle()) # True
print('var_1'.isidentifier())  # True - valid Python name
print('123'.isdecimal())    # True
print('123'.isnumeric())    # True (also handles fractions)
print('abc'.isprintable())  # True

Modify Methods

Strip whitespace/chars, replace substrings.

s = '  hello world  '
print(repr(s.strip()))        # 'hello world'
print(repr(s.lstrip()))       # 'hello world  '
print(repr(s.rstrip()))       # '  hello world'
print('***hi***'.strip('*'))  # hi

print('hello'.replace('l', 'r'))      # herro
print('hello'.replace('l', 'r', 1))   # herlo (max 1 replace)

# Python 3.9+
print('hello world'.removeprefix('hello '))  # world
print('hello world'.removesuffix(' world'))  # hello
Output
'hello world'
'hello world  '
'  hello world'
hi
herro
herlo
world
hello

Split & Join

Split strings into lists; join lists into strings.

s = 'a,b,c,d'
print(s.split(','))          # ['a', 'b', 'c', 'd']
print(s.split(',', 2))       # ['a', 'b', 'c,d'] (max 2 splits)
print(s.rsplit(',', 1))      # ['a,b,c', 'd']
print('hello\nworld'.splitlines())  # ['hello', 'world']

# join - reverse of split
words = ['Hello', 'World']
print(' '.join(words))   # Hello World
print('-'.join('abc'))   # a-b-c
Output
['a', 'b', 'c', 'd']
['a', 'b', 'c,d']
['a,b,c', 'd']
['hello', 'world']
Hello World
a-b-c

Align & Pad Methods

Align text within a field width.

print('hi'.center(10))        #     hi    
print('hi'.center(10, '-'))   # ----hi----
print('hi'.ljust(10, '.'))    # hi........
print('hi'.rjust(10, '.'))    # ........hi
print('42'.zfill(6))          # 000042
print('-42'.zfill(6))         # -00042
Output
    hi    
----hi----
hi........
........hi
000042
-00042

Format Methods

Multiple ways to format strings in Python.

name, age = 'Alice', 30

# f-strings (Python 3.6+) - preferred
print(f'Name: {name}, Age: {age}')
print(f'{3.14159:.2f}')   # 2 decimal places
print(f'{1000000:,}')     # comma separator
print(f'{42:08b}')        # binary, zero-padded
print(f'{name!r}')        # repr
print(f'{age=}')          # debug: prints age=30

# .format()
print('{} is {}'.format(name, age))
print('{0} and {0}'.format('echo'))
print('{n} is {a}'.format(n=name, a=age))
Output
Name: Alice, Age: 30
3.14
1,000,000
00101010
'Alice'
age=30
Alice is 30
echo and echo
Alice is 30

Partition & Translate

partition splits into exactly 3 parts; translate maps characters.

# partition
print('hello:world'.partition(':'))   # ('hello', ':', 'world')
print('hello world'.partition(':'))   # ('hello world', '', '')
print('a:b:c'.rpartition(':'))        # ('a:b', ':', 'c')

# translate + maketrans
table = str.maketrans('aeiou', '12345')
print('hello world'.translate(table))  # h2ll4 w4rld

# Delete chars with translate
table2 = str.maketrans('', '', 'aeiou')  # delete vowels
print('hello world'.translate(table2))   # hll wrld
Output
('hello', ':', 'world')
('hello world', '', '')
('a:b', ':', 'c')
h2ll4 w4rld
hll wrld

Encode/Decode

Convert between str and bytes.

s = 'hello'
b = s.encode('utf-8')      # str -> bytes
print(b)                   # b'hello'
print(type(b))             # bytes

decoded = b.decode('utf-8')  # bytes -> str
print(decoded)               # hello

# Unicode example
u = 'caf\u00e9'  # café
print(u.encode('utf-8'))   # b'caf\xc3\xa9'
Output
b'hello'
<class 'bytes'>
hello
b'caf\xc3\xa9'

Interview Mastery

Q: Why is string concatenation in a loop bad? What's the alternative?

A: Strings are immutable, so each `s += x` creates a new string object — O(n²) total. Use `''.join(list_of_strings)` which is O(n). Example: `result = ''.join(str(i) for i in range(1000))`.

Q: What's the difference between find() and index()?

A: `find()` returns -1 if substring not found. `index()` raises a `ValueError`. Use `find()` when absence is a valid case; use `index()` when absence means a bug.

Q: How do f-strings differ from .format()?

A: f-strings (3.6+) are evaluated at runtime and generally faster. They support expressions inline: `f'{2+2}'`. `.format()` is more flexible for templates stored as strings since f-strings must be defined inline. f-strings also support `=` for debugging: `f'{var=}'`.
