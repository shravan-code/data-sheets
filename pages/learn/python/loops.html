Section 2: Foundations

Loops

Basic for loop

Iterates over a sequence (range, list, string). The standard way to repeat a task a fixed number of times.

for i in range(3):
    print(f'Iteration {i}')
Output
Iteration 0
Iteration 1
Iteration 2

Loop with enumerate

Get both index and value simultaneously. Avoids manual counter variables.

fruits = ['apple', 'banana']
for i, fruit in enumerate(fruits):
    print(f'{i}: {fruit}')
Output
0: apple
1: banana

Loop with zip

Iterate over multiple iterables in parallel. Stops at the shortest iterable.

names = ['Alice', 'Bob']
scores = [90, 85]
for name, score in zip(names, scores):
    print(f'{name}: {score}')
Output
Alice: 90
Bob: 85

while loop

Repeats while condition is True. Use when iteration count is unknown.

# Basic while
n = 1
while n < 32:
    n *= 2
print(n)  # 32

# while with break
import random
random.seed(42)
while True:
    val = random.randint(1, 10)
    if val == 5:
        print(f'Found 5!')
        break
    print(val, end=' ')
Output
32
2 1 9 Found 5!

break, continue, pass

break exits loop; continue skips to next iteration; pass is a no-op placeholder.

# break
for i in range(10):
    if i == 5:
        break
    print(i, end=' ')
print()

# continue
for i in range(10):
    if i % 2 == 0:
        continue  # skip even
    print(i, end=' ')
print()

# pass - placeholder
def todo_function():
    pass  # implement later

# for-else (runs if loop completed without break)
for i in range(5):
    if i == 10:  # never true
        break
else:
    print('Loop completed without break')
Output
0 1 2 3 4 
1 3 5 7 9 
Loop completed without break

Iterating with enumerate, zip, reversed

Built-in helpers for common iteration patterns.

# enumerate with start index
for i, v in enumerate(['a','b','c'], start=1):
    print(i, v)

# zip stops at shortest
for a, b in zip([1,2,3], ['x','y']):
    print(a, b)

# reversed
for x in reversed([1,2,3,4]):
    print(x, end=' ')
print()

# zip_longest fills missing values
from itertools import zip_longest
for a, b in zip_longest([1,2,3], ['x'], fillvalue='?'):
    print(a, b)
Output
1 a
2 b
3 c
1 x
2 y
4 3 2 1 
1 x
2 ?
3 ?

Interview Mastery

Q: What is the for-else construct in Python?

A: The `else` block of a `for` or `while` loop runs only if the loop completes without hitting a `break`. Useful for search algorithms: loop to find item, break if found, else block handles 'not found' case.

Q: What's the difference between range() and list(range())?

A: `range()` returns a lazy range object (O(1) memory regardless of size). `list(range(n))` materializes it into a list (O(n) memory). Always prefer `range()` in for loops.
