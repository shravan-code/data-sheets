deque — Double-Ended Queue

All deque Methods

deque provides O(1) appends and pops from both ends. Lists are O(n) for left operations.

from collections import deque

# Creation
d = deque([1, 2, 3, 4, 5])
print(d)

# append/appendleft
d.append(6)
d.appendleft(0)
print(d)

# pop/popleft
d.pop()       # removes 6
d.popleft()   # removes 0
print(d)

# extend/extendleft
d.extend([6, 7])
d.extendleft([-1, -2])  # note: each item appended left -> reversed!
print(d)

# rotate(n) - right rotation if n>0, left if n<0
d2 = deque([1,2,3,4,5])
d2.rotate(2)
print(d2)   # [4,5,1,2,3]
d2.rotate(-2)
print(d2)   # back to [1,2,3,4,5]

# maxlen - sliding window
window = deque(maxlen=3)
for i in range(6):
    window.append(i)
    print(list(window))

# Other methods
d3 = deque([1,2,3,2,1])
print(d3.count(2))         # 2
print(d3.index(3))         # 2
d3.insert(2, 99)
print(d3)
d3.remove(99)
d3.reverse()
print(d3)

Interview Mastery

Q:

When use deque over list?

A:

Use deque when you need O(1) appendleft/popleft (list is O(n) for these). Typical use cases: queues (FIFO), sliding windows with maxlen, breadth-first search, implementing undo history. For random access by index, list is still faster.
