Dictionaries — All Methods

Dictionaries are hash maps: O(1) average for get/set/delete. Ordered by insertion since Python 3.7.

Accessing Data: get & setdefault

get() provides safe access with a default value to avoid KeyError. setdefault() returns the value if present, otherwise inserts the default.

d = {'a': 1}
print(d.get('b', 0))    # 0
print(d.setdefault('b', 2)) # 2
print(d)                # {'a': 1, 'b': 2}

Modifying: update, pop, popitem

update() merges another dict. pop() removes by key. popitem() removes the last inserted item (Python 3.7+).

d = {'a': 1, 'b': 2}
d.update({'c': 3, 'a': 0})
print(d.pop('b'))       # 2
print(d.popitem())      # ('c', 3)
print(d)                # {'a': 0}

Dict Views: keys, values, items

Return dynamic view objects that reflect changes in the dictionary. items() is perfect for iterating through pairs.

d = {'a': 1, 'b': 2}
print(list(d.keys()))   # ['a', 'b']
print(list(d.values())) # [1, 2]
for k, v in d.items():
    print(f'{k}:{v}', end=' ')
Output
['a', 'b']
[1, 2]
a:1 b:2

Dict Merging (Python 3.9+)

New operators for merging dicts.

d1 = {'a': 1, 'b': 2}
d2 = {'b': 3, 'c': 4}

# | operator - merge, right takes priority
merged = d1 | d2
print(merged)   # {'a':1, 'b':3, 'c':4}

# |= update in-place
d1 |= d2
print(d1)

# Old way
merged2 = {**d1, **d2}  # still works
print(merged2)
Output
{'a': 1, 'b': 3, 'c': 4}
{'a': 1, 'b': 3, 'c': 4}
{'a': 1, 'b': 3, 'c': 4}

defaultdict

Dict that auto-initializes missing keys with a factory function.

from collections import defaultdict

# Group by first letter
words = ['apple', 'avocado', 'banana', 'blueberry', 'cherry']
groups = defaultdict(list)
for word in words:
    groups[word[0]].append(word)
print(dict(groups))

# Count with default 0
counts = defaultdict(int)
for word in words:
    counts[word[0]] += 1
print(dict(counts))

# Nested defaultdict
tree = defaultdict(lambda: defaultdict(int))
tree['fruits']['apple'] += 1
tree['fruits']['banana'] += 2
print(dict(tree['fruits']))
Output
{'a': ['apple', 'avocado'], 'b': ['banana', 'blueberry'], 'c': ['cherry']}
{'a': 2, 'b': 2, 'c': 1}
{'apple': 1, 'banana': 2}

Counter

Specialized dict for counting hashable objects.

from collections import Counter

# Count characters
c = Counter('abracadabra')
print(c)
print(c.most_common(3))
print(c['a'])
print(c['z'])  # returns 0, not KeyError

# Count words
words = ['apple', 'banana', 'apple', 'cherry', 'banana', 'apple']
wc = Counter(words)
print(wc.most_common(2))

# Arithmetic on counters
c1 = Counter({'a': 3, 'b': 2})
c2 = Counter({'a': 1, 'b': 4, 'c': 1})
print(c1 + c2)   # add counts
print(c1 - c2)   # subtract, keep positives
print(c1 & c2)   # intersection (min)
print(c1 | c2)   # union (max)

# elements() - iterate with repetition
print(list(Counter({'a':2,'b':1}).elements()))

Interview Mastery

Q:

What is the time complexity of dict operations?

A:

Average O(1) for get, set, delete, and `in` operator. Worst case O(n) due to hash collisions (rare). Iteration is O(n).

Q:

How would you count word frequency in a sentence?

A:

Use Counter: `from collections import Counter; freq = Counter(sentence.split())`. Or manually: `freq = {}; freq[word] = freq.get(word, 0) + 1`.
