Conditionals

if / elif / else

Basic branching. Python uses indentation for blocks.

x = 15
if x > 20:
    print('big')
elif x > 10:
    print('medium')
else:
    print('small')
Output
medium

Ternary Expression

Inline conditional: value_if_true if condition else value_if_false

age = 20
status = 'adult' if age >= 18 else 'minor'
print(status)

# Nested ternary (avoid for readability)
grade = 85
result = 'A' if grade >= 90 else 'B' if grade >= 80 else 'C'
print(result)
Output
adult
B

match / case (Python 3.10+)

Structural pattern matching — more powerful than switch/case in other languages.

command = 'quit'
match command:
    case 'quit':
        print('Quitting')
    case 'start':
        print('Starting')
    case _:
        print('Unknown command')

# Match with destructuring
point = (1, 0)
match point:
    case (0, 0):
        print('Origin')
    case (x, 0):
        print(f'On x-axis at {x}')
    case (0, y):
        print(f'On y-axis at {y}')
    case (x, y):
        print(f'Point at ({x},{y})')
Output
Quitting
On x-axis at 1

Truthy and Falsy Values

In Python, many objects are implicitly False in boolean context.

# Falsy values
falsy = [False, 0, 0.0, '', [], {}, set(), None, 0j]
for val in falsy:
    print(f'{repr(val):10} -> {bool(val)}')

# Truthy: everything else
print(bool(42))     # True
print(bool([0]))    # True - non-empty list
Output
False      -> False
0          -> False
0.0        -> False
''         -> False
[]         -> False
{}         -> False
set()      -> False
None       -> False
0j         -> False
True
True

Interview Mastery

Q:

What is the difference between `if x:` and `if x is not None:`?

A:

`if x:` is False for None, 0, '', [], {} etc. `if x is not None:` is only False for None. Use the latter when 0 or empty string is a valid value you want to keep.
