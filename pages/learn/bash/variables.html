2. Variables

Variables are assigned without spaces around =. Access them with $ prefix. Variable names are case-sensitive.

Declaring and Using Variables


Variables are assigned without spaces around =. Access them with $ prefix. Variable names are case-sensitive.


Basic Variable Assignment


name="Alice"
age=30
echo "Name: $name, Age: $age"

Curly Brace Syntax


fruit="apple"
echo "I have ${fruit}s"


Variable Types


Bash variables are untyped by default, but you can use declare to define types like integers or arrays.


Integer Declaration


declare -i count=10
count+=5
echo $count  # Output: 15

Readonly Variable


readonly PI=3.14159
echo $PI
# PI=3 would cause an error

Lowercase Variable


declare -l lowered="HELLO"
echo $lowered  # Output: hello


Environment Variables


Environment variables are available to the script and child processes. Use export to pass them to subprocesses.


Common Environment Variables


echo $HOME      # /home/username
echo $USER      # username
echo $PATH      # /usr/bin:/bin:...
echo $PWD       # Current directory
echo $SHELL     # /bin/bash
echo $HOSTNAME  # machine-name

Exporting Variables


export MY_VAR="hello"
bash -c 'echo $MY_VAR'  # Output: hello


Special Variables


Bash provides special variables for script arguments, process info, and status codes.


Script Arguments


#!/bin/bash
echo "Script name: $0"
echo "First arg: $1"
echo "Second arg: $2"
echo "All args: $@"
echo "Number of args: $#"

Exit Status and PID


ls /nonexistent 2>/dev/null
echo "Exit status: $?"  # 1 (error)
echo "Current PID: $$"
echo "Last background PID: $!"


Variable Substitution & Defaults


Bash provides powerful parameter expansion to set defaults, substrings, and transformations.


Default Values


name=""
echo ${name:-"Default"}   # Default (if unset or empty)
echo ${name:="Assigned"}  # Assigns and prints
echo ${name:?"Error!"}    # Error if unset

Substring Extraction


str="Hello, World!"
echo ${str:7:5}   # World
echo ${str:0:5}   # Hello
echo ${#str}      # 13 (length)

Pattern Removal


file="/home/user/docs/report.txt"
echo ${file##*/}   # report.txt (basename)
echo ${file%/*}    # /home/user/docs (dirname)
echo ${file%.txt}  # /home/user/docs/report
