8. String Manipulation

These are the three pillars of text processing in Bash. grep searches, sed transforms, awk processes structured text.

grep, sed, awk


These are the three pillars of text processing in Bash. grep searches, sed transforms, awk processes structured text.


grep Patterns


# Basic search
grep "error" logfile.txt

# Case-insensitive
grep -i "error" logfile.txt

# Count matches
grep -c "error" logfile.txt

# Show line numbers
grep -n "error" logfile.txt

# Invert match (exclude)
grep -v "debug" logfile.txt

# Regex
grep -E "^[0-9]{4}-[0-9]{2}" dates.txt

sed Transformations


# Replace text
sed 's/old/new/g' file.txt

# Delete lines matching pattern
sed '/error/d' file.txt

# Print specific lines
sed -n '5,10p' file.txt

# Add prefix to each line
sed 's/^/PREFIX: /' file.txt

# In-place edit
sed -i 's/foo/bar/g' file.txt

awk Processing


# Print columns
awk '{print $1, $3}' data.txt

# Field separator
awk -F':' '{print $1}' /etc/passwd

# Conditional
awk '$3 > 1000 {print $1, $3}' data.txt

# Sum a column
awk '{sum += $2} END {print "Total:", sum}' data.txt

# Print with formatting
awk '{printf "%-20s %5d\n", $1, $2}' data.txt


cut, tr, sort, uniq


Utility commands for column extraction, character translation, sorting, and deduplication.


cut & tr


# Extract columns
cut -d':' -f1,3 /etc/passwd

# Extract characters
cut -c1-5 file.txt

# Translate characters
echo "Hello World" | tr '[:lower:]' '[:upper:]'

# Delete characters
echo "H-e-l-l-o" | tr -d '-'

sort & uniq


# Sort alphabetically
sort names.txt

# Sort numerically
sort -n numbers.txt

# Sort in reverse
sort -r file.txt

# Sort by field
sort -t',' -k2 data.csv

# Remove duplicates
sort names.txt | uniq

# Count occurrences
sort names.txt | uniq -c | sort -rn
