File & Directory Operations

File Management

Bash provides robust file manipulation through built-in commands.

BASH
touch newfile.txt              # create empty file
cp source.txt dest.txt         # copy file
cp -r src_dir/ dst_dir/        # copy directory
mv oldname.txt newname.txt     # rename/move
rm file.txt                    # delete file
rm -rf directory/              # delete directory (CAREFUL!)
ln -s target link_name         # create symlink

Directory Operations

Creating, navigating, and managing directories.

BASH
mkdir mydir                  # create directory
mkdir -p a/b/c               # create nested dirs
rmdir emptydir               # remove empty dir
pushd /tmp                   # save & change dir
popd                         # restore previous dir
ls -la                       # list with details
tree -L 2                    # tree view (2 levels)

File Permissions

Linux permissions follow owner/group/other model. Use chmod (symbolic or octal) and chown.

BASH
chmod 755 script.sh    # rwxr-xr-x
chmod 644 file.txt     # rw-r--r--
chmod +x script.sh     # add execute
chmod -w file.txt      # remove write
chmod u+x,g+x file.sh  # symbolic
