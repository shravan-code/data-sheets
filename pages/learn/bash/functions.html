7. Functions

Functions group reusable code. They can accept arguments and return values via exit codes or stdout.

Defining and Calling Functions


Functions group reusable code. They can accept arguments and return values via exit codes or stdout.


Basic Function


greet() {
  echo "Hello, $1!"
}
greet "Alice"   # Hello, Alice!
greet "Bob"     # Hello, Bob!

Function with Return Value


add() {
  echo $(($1 + $2))
}
result=$(add 5 10)
echo "Sum: $result"  # Sum: 15

Function with Exit Code


is_even() {
  [ $(($1 % 2)) -eq 0 ]
}
if is_even 4; then
  echo "4 is even"
fi


Local Variables


Use local to scope variables inside functions, preventing them from leaking into the global scope.


Local Scope


x=10
my_func() {
  local x=99
  echo "Inside: $x"  # 99
}
my_func
echo "Outside: $x"  # 10 (unchanged)


Recursive Functions


Bash supports recursion, but it is slow and can hit limits. Useful for tree traversal and mathematical problems.


Factorial


factorial() {
  if [ $1 -le 1 ]; then
    echo 1
  else
    local prev=$(factorial $(($1 - 1)))
    echo $(($1 * $prev))
  fi
}
echo $(factorial 5)  # 120

Fibonacci


fibonacci() {
  if [ $1 -le 1 ]; then
    echo $1
  else
    local a=$(fibonacci $(($1 - 1)))
    local b=$(fibonacci $(($1 - 2)))
    echo $((a + b))
  fi
}
for i in {0..9}; do printf "$(fibonacci $i) "; done


Function Libraries


Source external files to include reusable function libraries with the source or . command.


Create and Source a Library


# lib.sh
log_info()  { echo "[INFO]  $(date) $1"; }
log_error() { echo "[ERROR] $(date) $1" >&2; }
log_warn()  { echo "[WARN]  $(date) $1"; }

# main.sh
source ./lib.sh
log_info "Script started"
log_warn "Low disk space"
