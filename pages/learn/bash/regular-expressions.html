Regular Expressions

Basic & Extended Regex

Regular expressions are patterns used with grep, sed, and awk for text matching and extraction.

BASH
# Match start of line
grep '^root' /etc/passwd

# Match end of line
grep 'bash$' /etc/passwd

# Match any character
grep 'c.t' words.txt  # cat, cot, cut

# Quantifiers
grep -E 'colou?r' text.txt    # color or colour
grep -E '[0-9]+' text.txt     # one or more digits
grep -E '[A-Za-z]{3,6}' text  # 3 to 6 letters

# Character classes
grep -E '^[[:alpha:]]+$' words   # only letters
grep -E '^[[:digit:]]+$' nums    # only digits
