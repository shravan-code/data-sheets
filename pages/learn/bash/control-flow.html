6. Control Flow

Conditional branching using if/elif/else. The condition can be any command—its exit status determines the branch.

if / elif / else


Conditional branching using if/elif/else. The condition can be any command—its exit status determines the branch.


Basic if-else


#!/bin/bash
read -p "Enter age: " age
if [ $age -ge 18 ]; then
  echo "Adult"
elif [ $age -ge 13 ]; then
  echo "Teenager"
else
  echo "Child"
fi

if with Command


if grep -q "error" logfile.txt; then
  echo "Errors found in log"
fi


case Statement


case matches a variable against patterns, like a switch statement. Patterns support wildcards.


Basic case


read -p "Enter day: " day
case $day in
  Monday|Tuesday|Wednesday|Thursday|Friday)
    echo "Weekday"
    ;;
  Saturday|Sunday)
    echo "Weekend"
    ;;
  *)
    echo "Unknown day"
    ;;
esac

case with Options


option="$1"
case "$option" in
  -h|--help)   echo "Show help" ;;
  -v|--version) echo "Version 1.0" ;;
  -q|--quiet)  QUIET=true ;;
  *)  echo "Unknown option: $option" ; exit 1 ;;
esac


for Loop


Iterate over lists, ranges, or command output using for loops.


List Loop


for animal in cat dog bird; do
  echo "Animal: $animal"
done

Range with seq / brace


for i in {1..5}; do
  echo "Count: $i"
done

for i in $(seq 0 2 10); do  # 0,2,4,6,8,10
  echo $i
done

C-Style for Loop


for ((i=0; i<5; i++)); do
  echo "Index: $i"
done

Loop Over Files


for file in *.txt; do
  echo "Processing: $file"
done


while Loop


while loops execute as long as the condition is true. Use break and continue to control flow.


Basic while


count=1
while [ $count -le 5 ]; do
  echo "Count: $count"
  ((count++))
done

Read File Line by Line


while IFS= read -r line; do
  echo "Line: $line"
done < file.txt

Infinite Loop with Break


while true; do
  read -p "Enter 'quit' to exit: " input
  [ "$input" = "quit" ] && break
  echo "You entered: $input"
done


until Loop


until is the inverse of while—it executes until the condition becomes true.


Basic until


num=1
until [ $num -gt 5 ]; do
  echo "Num: $num"
  ((num++))
done


select Menu


select creates interactive numbered menus from a list of options.


Interactive Menu


PS3="Choose an option: "
select opt in "Start" "Stop" "Restart" "Quit"; do
  case $opt in
    "Start")   echo "Starting..." ;;
    "Stop")    echo "Stopping..." ;;
    "Restart") echo "Restarting..." ;;
    "Quit")    break ;;
    *) echo "Invalid option" ;;
  esac
done
