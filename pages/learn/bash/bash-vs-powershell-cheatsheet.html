17. Bash vs PowerShell Cheatsheet

Side-by-side comparison of Bash (Linux/macOS) and PowerShell (Windows/cross-platform) commands and syntax

Basic Output


Task Bash PowerShell
Print text echo "Hello, World!" Write-Host "Hello, World!"
Print without newline echo -n "No newline" Write-Host -NoNewline "No newline"
Print to stderr echo "Error" >&2 Write-Error "Error"
Formatted output printf "%-10s %5d\n" "item" 42 "{0,-10} {1,5}" -f "item", 42

Variables


Task Bash PowerShell
Assign variable name="Alice" $name = "Alice"
Use variable echo $name Write-Host $name
Integer variable declare -i num=10 [int]$num = 10
Read-only variable readonly PI=3.14 Set-Variable -Name PI -Value 3.14 -Option Constant
Unset variable unset name Remove-Variable name
String length ${#str} $str.Length

Conditionals


Task Bash PowerShell
if-else if [ $x -gt 5 ]; then echo "yes"; fi if ($x -gt 5) { Write-Host "yes" }
Equal comparison (int) [ $a -eq $b ] $a -eq $b
Not equal (int) [ $a -ne $b ] $a -ne $b
Greater than [ $a -gt $b ] $a -gt $b
String equals [ "$a" = "$b" ] $a -eq $b
String contains [[ $str == *substring* ]] $str -like "*substring*"
Regex match [[ $str =~ pattern ]] $str -match "pattern"
Null/empty check [ -z "$str" ] [string]::IsNullOrEmpty($str)

Loops


Task Bash PowerShell
For range loop for i in {1..10}; do echo $i; done for ($i=1; $i -le 10; $i++) { Write-Host $i }
For each item for item in a b c; do echo $item; done foreach ($item in @('a','b','c')) { Write-Host $item }
While loop while [ $x -lt 10 ]; do ((x++)); done while ($x -lt 10) { $x++ }
Loop with break while true; do break; done while ($true) { break }
Loop over files for f in *.txt; do echo $f; done Get-ChildItem *.txt | ForEach-Object { Write-Host $_.Name }

Functions


Task Bash PowerShell
Define function greet() { echo "Hello $1"; } function Greet($name) { Write-Host "Hello $name" }
Call function greet "Alice" Greet "Alice"
Return value result=$(add 5 3) $result = Add 5 3
Local variable local x=10 $local:x = 10

File & Directory


Task Bash PowerShell
List files ls -la Get-ChildItem -Force
Change directory cd /path/to/dir Set-Location /path/to/dir
Current directory pwd Get-Location
Create directory mkdir -p dir/subdir New-Item -ItemType Directory -Path dir/subdir
Copy file cp source.txt dest.txt Copy-Item source.txt dest.txt
Move file mv old.txt new.txt Move-Item old.txt new.txt
Delete file rm file.txt Remove-Item file.txt
File exists? [ -f file.txt ] Test-Path file.txt
Read file cat file.txt Get-Content file.txt
Write to file echo "text" > file.txt "text" | Out-File file.txt
Append to file echo "text" >> file.txt "text" | Add-Content file.txt
Find files find . -name "*.txt" Get-ChildItem -Recurse -Filter *.txt

String Operations


Task Bash PowerShell
Substring ${str:2:5} $str.Substring(2,5)
Replace ${str/old/new} $str -replace 'old','new'
Uppercase ${str^^} $str.ToUpper()
Lowercase ${str,,} $str.ToLower()
Split string IFS=',' read -ra arr <<< "$str" $arr = $str -split ','
Join array IFS=','; echo "${arr[*]}" $arr -join ','
Trim whitespace echo $str | xargs $str.Trim()

Arrays


Task Bash PowerShell
Create array arr=(1 2 3) $arr = @(1, 2, 3)
Access element ${arr[0]} $arr[0]
Array length ${#arr[@]} $arr.Count
All elements ${arr[@]} $arr
Append element arr+=(4) $arr += 4
Array slice ${arr[@]:1:3} $arr[1..3]
Loop over array for x in "${arr[@]}"; do echo $x; done foreach ($x in $arr) { Write-Host $x }

Input / Output


Task Bash PowerShell
Read user input read -p "Name: " name $name = Read-Host "Name"
Read password read -s -p "Pass: " pass $pass = Read-Host -AsSecureString "Pass"
Redirect stdout cmd > out.txt cmd | Out-File out.txt
Redirect stderr cmd 2> err.txt cmd 2> err.txt
Redirect both cmd &> all.txt cmd *> all.txt
Pipe cmd1 | cmd2 cmd1 | cmd2
Discard output cmd > /dev/null 2>&1 cmd | Out-Null

Process Management


Task Bash PowerShell
List processes ps aux Get-Process
Kill process by PID kill 1234 Stop-Process -Id 1234
Kill by name pkill nginx Stop-Process -Name nginx
Run in background cmd & Start-Job { cmd }
Wait for background wait Get-Job | Wait-Job
Get PID echo $$ $PID

Environment & System


Task Bash PowerShell
Set env variable export MY_VAR=value $env:MY_VAR = 'value'
Get env variable echo $HOME $env:USERPROFILE
List all env vars env Get-ChildItem Env:
Current user whoami $env:USERNAME
Hostname hostname $env:COMPUTERNAME
OS info uname -a $PSVersionTable.OS
Date/time date Get-Date
Formatted date date +%Y-%m-%d Get-Date -Format 'yyyy-MM-dd'

Error Handling


Task Bash PowerShell
Exit on error set -e $ErrorActionPreference = 'Stop'
Get exit code echo $? $LASTEXITCODE
Custom exit code exit 1 exit 1
Try-catch if ! cmd; then echo "failed"; fi try { cmd } catch { Write-Error $_ }
Trap signal trap 'cleanup' EXIT INT TERM Register-EngineEvent -SourceIdentifier PowerShell.Exiting -Action { cleanup }

Network & Web


Task Bash PowerShell
Download file curl -O https://example.com/file.zip Invoke-WebRequest -Uri https://example.com/file.zip -OutFile file.zip
HTTP GET request curl https://api.example.com/data Invoke-RestMethod -Uri https://api.example.com/data
HTTP POST (JSON) curl -X POST -H 'Content-Type: application/json' -d '{"key":"val"}' url Invoke-RestMethod -Method Post -Uri url -Body (@{key='val'} | ConvertTo-Json) -ContentType 'application/json'
Test connectivity ping -c 4 google.com Test-NetConnection -ComputerName google.com
