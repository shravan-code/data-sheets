Section 20: Views

Updatable View

Views that allow INSERT/UPDATE/DELETE. Possible only for simple views (no joins, aggregates, DISTINCT, or GROUP BY).

Syntax Pattern

-- Simple updatable view:
CREATE VIEW active_employees AS
SELECT emp_id, name, salary FROM employees WHERE status = 'active';

-- This UPDATE works because the view is simple:
UPDATE active_employees SET salary = 80000 WHERE emp_id = 1;
Output
Execution result varies based on data.

Creating a Updatable View

CREATE UPDATABLE VIEW active_users_view AS 
SELECT * FROM users WHERE status = 'active';
Output
Virtual table created. You can now query 'active_users_view' like a normal table.
Pro Tips & Tricks
  • Always verify your logic with a small dataset first
  • Use EXPLAIN to see how the database handles your query
