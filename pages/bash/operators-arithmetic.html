5. Operators & Arithmetic

Bash supports integer arithmetic via (( )) or $(()), and floating-point via bc or awk.

Arithmetic Operations


Bash supports integer arithmetic via (( )) or $(()), and floating-point via bc or awk.


Integer Arithmetic


a=10; b=3
echo $((a + b))   # 13
echo $((a - b))   # 7
echo $((a * b))   # 30
echo $((a / b))   # 3 (integer)
echo $((a % b))   # 1 (modulo)
echo $((a ** b))  # 1000 (power)

Increment & Decrement


x=5
((x++))
echo $x   # 6
((x+=10))
echo $x   # 16
((x--))
echo $x   # 15

Floating Point with bc


result=$(echo "scale=4; 10 / 3" | bc)
echo $result   # 3.3333


Comparison Operators


Integer comparisons use -eq, -ne, -lt, -le, -gt, -ge. String comparisons use =, !=, <, >.


Integer Comparisons


a=10; b=20
[ $a -eq $b ] && echo "equal" || echo "not equal"
[ $a -lt $b ] && echo "a less than b"
[ $a -ge 10 ] && echo "a >= 10"

String Comparisons


s1="hello"; s2="world"
[ "$s1" = "$s2" ]  && echo "same"
[ "$s1" != "$s2" ] && echo "different"
[ -z "$s1" ]       && echo "empty"
[ -n "$s1" ]       && echo "not empty"

Modern [[ Double Brackets ]]


name="Alice"
[[ $name == A* ]] && echo "Starts with A"
[[ $name =~ ^[A-Z] ]] && echo "Starts with uppercase"


Logical Operators


Combine conditions using && (AND), || (OR), and ! (NOT).


AND and OR


age=25
[[ $age -ge 18 && $age -le 65 ]] && echo "Working age"
[[ $age -lt 18 || $age -gt 65 ]] && echo "Not working age"

NOT Operator


file="test.txt"
[[ ! -f "$file" ]] && echo "File does not exist"


File Test Operators


Test file existence, type, and permissions using various -x flags.


File Tests


file="test.txt"
[ -e "$file" ] && echo "exists"
[ -f "$file" ] && echo "is a regular file"
[ -d "$file" ] && echo "is a directory"
[ -r "$file" ] && echo "readable"
[ -w "$file" ] && echo "writable"
[ -x "$file" ] && echo "executable"
[ -s "$file" ] && echo "non-empty"
[ -L "$file" ] && echo "is a symlink"
