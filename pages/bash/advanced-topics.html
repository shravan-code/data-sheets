Advanced Topics

Command Substitution

Capture command output into a variable using $() or backticks.

Command Substitution

current_date=$(date +%Y-%m-%d)
file_count=$(ls | wc -l)
uptime_info=$(uptime)
echo "Date: $current_date, Files: $file_count"

Subshells

Code in ( ) runs in a subshell—a child process. Changes don't affect the parent.

Subshell Example

x=10
(x=99; echo "Inside subshell: $x")  # 99
echo "Outside subshell: $x"           # 10

Group Commands

# Braces run in current shell
{ date; ls; whoami; } > report.txt

eval and exec

eval executes a string as a command. exec replaces the current process with a new one.

eval Usage

cmd="echo Hello World"
eval "$cmd"  # Hello World

# Dynamic variable names
var_name="color"
eval "${var_name}=blue"
echo $color  # blue

exec Usage

# Replace shell with a command
exec python3 myapp.py
# Script stops here; python takes over PID

# Redirect all script output
exec > /tmp/output.log 2>&1
echo "This goes to the log"

Named Pipes (FIFOs)

Named pipes allow two-way communication between processes using a special file.

FIFO Example

mkfifo /tmp/mypipe

# Producer
echo "Hello from producer" > /tmp/mypipe &

# Consumer
cat /tmp/mypipe

rm /tmp/mypipe

coproc (Coprocesses)

coproc runs a command in the background and connects its stdin/stdout to file descriptors.

Coprocess

coproc bc
echo "3 + 4" >&${COPROC[1]}
read result <&${COPROC[0]}
echo "3 + 4 = $result"  # 7
