Debugging

Debugging Techniques

Bash offers several built-in debugging tools and strategies.

set -x Trace Mode

#!/bin/bash
set -x  # enable debug tracing
x=5
y=10
echo $((x + y))
set +x  # disable tracing

Run Script in Debug Mode

bash -x myscript.sh
bash -v myscript.sh  # print each line before execution

Debug Specific Block

echo "Before debug"
set -x
complex_function
set +x
echo "After debug"

Log to File

LOG_FILE="/tmp/script_debug.log"
exec 2>"$LOG_FILE"  # redirect stderr to log file
set -x
# all trace output goes to log file
