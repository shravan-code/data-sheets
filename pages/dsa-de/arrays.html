2. Arrays

Traversal, Two Pointers, Sliding Window, Prefix Sum.

Understanding Arrays

An Array is the most fundamental data structure. It stores elements in contiguous memory locations, meaning they sit right next to each other in your computer's RAM.

graph LR subgraph Array direction LR A["10 (Index 0)"] --- B["20 (Index 1)"] --- C["30 (Index 2)"] --- D["40 (Index 3)"] --- E["50 (Index 4)"] end

Core Operations

  • Access: O(1). Since items are contiguous, we can jump directly to any index.
  • Search: O(N). In an unsorted array, we must check every element one by one.
  • Insertion/Deletion: O(N). When you add or remove an item in the middle, you have to shift all subsequent items to make room or close the gap.

Advanced Techniques

Most array interview questions are variations of a few patterns. If you learn the pattern, you can solve a whole family of problems.

1) Traversal (Single pass)

When to use: You need to compute something over all elements: sum, min/max, counts, frequency of a value, check a condition, build a transformed list.

Key idea: Keep a running answer while scanning left → right. This gives you O(N) time and O(1) extra space (usually).

def count_even(nums):
    count = 0
    for x in nums:
        if x % 2 == 0:
            count += 1
    return count

# Input
nums = [3, 10, 7, 2, 9, 12]

# Output
print(count_even(nums))  # 3  (10, 2, 12)

2) Two Pointers

When to use: Problems involving pairs, reversing, partitioning, or searching in a sorted array. Common signals: “two numbers”, “closest”, “palindrome-like”, “sorted array”, “remove duplicates in-place”.

Key idea: Maintain two indices (often l and r) and move one or both pointers based on a condition. This often converts a nested loop into a single pass.

def two_sum_sorted(nums, target):
    l, r = 0, len(nums) - 1
    while l < r:
        s = nums[l] + nums[r]
        if s == target:
            return (l, r)
        if s < target:
            l += 1
        else:
            r -= 1
    return None

# Input (sorted array)
nums = [1, 2, 4, 7, 11, 15]
target = 15

# Output
print(two_sum_sorted(nums, target))  # (2, 3) because 4 + 11 = 15

3) Sliding Window

When to use: “subarray/substring”, “contiguous”, “largest/smallest window”, “at most / at least K”, “maximum sum/average of size K”.

Key idea: Keep a window [l..r] and update its state when you expand r (add) and shrink l (remove). Avoid recomputing sums/counts from scratch.

Fixed-size window: maximum sum of size K
def max_sum_subarray(nums, k):
    window_sum = sum(nums[:k])
    best = window_sum

    for r in range(k, len(nums)):
        window_sum += nums[r] - nums[r - k]
        best = max(best, window_sum)

    return best

# Input
nums = [2, 1, 5, 1, 3, 2]
k = 3

# Output
print(max_sum_subarray(nums, k))  # 9  (5 + 1 + 3)
Variable-size window: smallest subarray length with sum ≥ S
def min_len_subarray_at_least_s(nums, s):
    l = 0
    window_sum = 0
    best = float('inf')

    for r in range(len(nums)):
        window_sum += nums[r]
        while window_sum >= s:
            best = min(best, r - l + 1)
            window_sum -= nums[l]
            l += 1

    return 0 if best == float('inf') else best

# Input
nums = [2, 3, 1, 2, 4, 3]
s = 7

# Output
print(min_len_subarray_at_least_s(nums, s))  # 2  ([4, 3])

4) Prefix Sum

When to use: Many “range sum” queries, subarray sum equals K, “sum between i and j” repeatedly, or “count subarrays with property …”.

Key idea: Build a prefix array where pref[i] = sum of first i elements. Then range sum becomes:

\(\text{sum}(l..r) = \text{pref}[r+1] - \text{pref}[l]\)

Range sum queries in O(1)
def build_prefix(nums):
    pref = [0]
    for x in nums:
        pref.append(pref[-1] + x)
    return pref

def range_sum(pref, l, r):
    return pref[r + 1] - pref[l]

# Input
nums = [5, -2, 3, 1, 2]
pref = build_prefix(nums)

# Output
print(range_sum(pref, 1, 3))  # 2  (-2 + 3 + 1)
print(range_sum(pref, 0, 4))  # 9  (sum of all)
Subarray sum equals K (prefix sum + hashmap)
def count_subarrays_sum_k(nums, k):
    freq = {0: 1}  # prefix_sum -> count
    prefix = 0
    count = 0

    for x in nums:
        prefix += x
        count += freq.get(prefix - k, 0)
        freq[prefix] = freq.get(prefix, 0) + 1

    return count

# Input
nums = [1, 2, 3, -2, 5, -3, 1]
k = 3

# Output
print(count_subarrays_sum_k(nums, k))  # 5
