3. Strings

String Manipulation, Palindromes, Anagrams.

String Manipulation

A string is essentially an array of characters. However, in many modern languages like Python, strings are immutable—meaning once they are created, they cannot be changed.

Key Concepts

  • Immutability: In Python, s = "Hello" followed by s += "!" actually creates a brand new string in memory.
  • Slicing: Extracting parts of a string (e.g., s[1:4]) is a core operation.
  • Character Frequency: Using HashMaps to count occurrences is the most common interview pattern.

Common Patterns

  • Palindrome: A word that reads the same forwards and backwards.
  • Anagram: Two strings containing the exact same characters in any order.

Python Example: Palindrome & Anagram

# Check if a string is a palindrome using Two Pointers
def is_palindrome(s):
    left, right = 0, len(s) - 1
    while left < right:
        if s[left] != s[right]: return False
        left, right = left + 1, right - 1
    return True

# Check if two strings are anagrams using Counter
from collections import Counter
def is_anagram(s1, s2):
    return Counter(s1) == Counter(s2)

Detailed Concepts (with Code & I/O)

1) Immutability (Transform by creating a new string)

In Python: Strings are immutable. So when you do something like s += "!", Python creates a new string instead of modifying the old one.

Common interview implication: Build the answer incrementally using a list, then join (or use comprehensions/slicing) instead of trying to mutate characters in place.

def reverse_string(s):
    # We create a NEW string (valid because strings are immutable).
    return ''.join(reversed(s))

# Input
s = "data"

# Output
print(reverse_string(s))  # "atad"

2) Slicing & Indexing (extract substrings fast)

Slicing rule: s[l:r] includes index l but excludes index r.

Why it shows up: Many problems ask you to normalize strings (remove prefix/suffix), compare parts, or build substrings.

def extract_part(s):
    # Extract indices [2..5] => s[2:6]
    return s[2:6]

# Input
s = "cakedata"

# Output
print(extract_part(s))  # "keda"

3) Character Frequency (HashMap / Counter pattern)

Use it when: you need to compare counts (anagrams, grouping, “same characters”, isomorphic checks).

def freq_map(s):
    freq = {}
    for ch in s:
        freq[ch] = freq.get(ch, 0) + 1
    return freq

# Input
s = "abracadabra"

# Output
print(freq_map(s)['a'])  # 5

4) Palindrome (Two Pointers)

Idea: Put two pointers at the ends. If characters mismatch, it cannot be a palindrome.

Complexity: O(N) time, O(1) extra space.

def is_palindrome(s):
    left, right = 0, len(s) - 1
    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1
    return True

# Input
s = "racecar"

# Output
print(is_palindrome(s))  # True

5) Anagram (frequency match)

Definition: Two strings are anagrams if they contain the same characters with the same counts (order doesn’t matter).

from collections import Counter

def is_anagram(s1, s2):
    if len(s1) != len(s2):
        return False
    return Counter(s1) == Counter(s2)

# Input
s1 = "listen"
s2 = "silent"

# Output
print(is_anagram(s1, s2))  # True
