4. Operators

PowerShell provides a rich set of operators for comparison, logical operations, string manipulation, and more.

Comparison Operators

PowerShell comparison operators are case-insensitive by default. Prefix with 'c' for case-sensitive and 'i' for explicit case-insensitive.

All Comparison Operators

# Equality
5 -eq 5       # Equal: True
5 -ne 3       # Not Equal: True

# Relational
10 -gt 5      # Greater Than: True
3 -lt 8       # Less Than: True
5 -ge 5       # Greater or Equal: True
4 -le 4       # Less or Equal: True

# String matching (case-insensitive by default)
'Hello' -eq 'hello'    # True
'Hello' -ceq 'hello'   # False (case-sensitive)

# Pattern matching
'PowerShell' -like 'Power*'     # Wildcard: True
'PowerShell' -notlike 'Bash*'   # True
'hello123' -match '^[a-z]+\d+$' # Regex: True
'hello123' -notmatch '^\d+$'    # True

# Containment
3 -in 1,2,3,4        # True
5 -notin 1,2,3       # True
1,2,3 -contains 2    # True

# Type check
$x = 42
$x -is [int]         # True
$x -isnot [string]   # True
$x -as [string]      # Converts to '42'

Logical Operators

Logical operators combine boolean expressions.

Logical Operators

$a = $true
$b = $false

$a -and $b   # AND: False
$a -or $b    # OR: True
-not $a      # NOT: False
!$a          # NOT shorthand: False
$a -xor $b   # XOR: True

# Short-circuit evaluation
$x = $null
if ($x -ne $null -and $x.Length -gt 0) {
    Write-Host 'Not null and not empty'
}

Arithmetic & Assignment Operators

Standard math and assignment operators with some PowerShell-specific additions.

Arithmetic Operators

5 + 3    # 8
10 - 4   # 6
3 * 4    # 12
15 / 4   # 3.75
15 % 4   # Modulo: 3

# Assignment operators
$x = 10
$x += 5   # 15
$x -= 3   # 12
$x *= 2   # 24
$x /= 4   # 6
$x %= 4   # 2

# Increment / Decrement
$x++
$x--
++$x
--$x

# String and array concatenation
'Hello' + ' ' + 'World'
@(1,2) + @(3,4)   # 1,2,3,4
'Ha' * 3           # HaHaHa

Special Operators

PowerShell has several unique operators for pipeline manipulation, range, and redirection.

Special Operators

# Range operator
1..10           # 1 to 10
10..1           # 10 to 1
'a'..'z'        # a to z

# Call operator
$cmd = 'Get-Process'
& $cmd          # Executes Get-Process

# Dot sourcing
. .\myScript.ps1  # Runs script in current scope

# Subexpression
"Files: $(Get-ChildItem | Measure-Object | Select-Object -Exp Count)"

# Array subexpression
@(Get-Process | Where-Object CPU -gt 10)

# Join operator
-join ('a','b','c')        # abc
'a','b','c' -join '-'      # a-b-c

# Split operator
'a,b,c' -split ','         # 'a', 'b', 'c'
'hello world' -split ' '

# Redirection
Get-Process > processes.txt         # Overwrite
Get-Process >> processes.txt        # Append
Get-Process 2> errors.txt          # Stderr
Get-Process *> all_output.txt      # All streams

# Ternary (PowerShell 7+)
$x = 10
$result = $x -gt 5 ? 'big' : 'small'
