9. File System & I/O

Working with files, directories, and data reading/writing.

Navigating the File System

PowerShell uses a provider model. The filesystem is the default, but the same commands work for registry, certificates, etc.

File System Navigation

Get-Location            # pwd
Set-Location C:\Users   # cd
Set-Location ..         # cd ..
Set-Location ~          # Home directory
Push-Location C:\Temp   # Push & navigate
Pop-Location            # Return to previous

Get-ChildItem           # ls / dir
Get-ChildItem -Recurse -Filter '*.txt'
Get-ChildItem -Hidden
Get-ChildItem -Directory
Get-ChildItem -File

# Test if path exists
Test-Path 'C:\Windows'
Test-Path 'C:\file.txt' -PathType Leaf
Test-Path 'C:\folder'   -PathType Container

File Operations

Creating, copying, moving, and deleting files and directories.

CRUD Operations

# Create
New-Item -Path 'C:\Temp\test.txt' -ItemType File
New-Item -Path 'C:\Temp\NewFolder' -ItemType Directory
New-Item -Path 'C:\Temp\test.txt' -ItemType File -Force  # Overwrite

# Copy
Copy-Item 'C:\source.txt' 'C:\dest.txt'
Copy-Item 'C:\Folder' 'D:\Backup' -Recurse

# Move / Rename
Move-Item 'C:\old.txt' 'C:\new.txt'
Rename-Item 'C:\file.txt' 'renamed.txt'

# Delete
Remove-Item 'C:\Temp\test.txt'
Remove-Item 'C:\TempFolder' -Recurse -Force

# Get file info
$file = Get-Item 'C:\Windows\explorer.exe'
$file.Name
$file.Length
$file.LastWriteTime
$file.CreationTime
$file.Extension
$file.FullName

Reading & Writing Files

Reading text, CSV, XML, and JSON files, and writing output to files.

Text File I/O

# Read all content
$content = Get-Content 'C:\log.txt'

# Read as single string
$text = Get-Content 'C:\log.txt' -Raw

# Read first/last N lines
Get-Content 'C:\big.log' -TotalCount 10
Get-Content 'C:\big.log' -Tail 20

# Write to file
Set-Content 'output.txt' 'Hello, World!'
Add-Content 'output.txt' 'New line'

# Redirect
'Line 1' | Out-File 'out.txt'
'Line 2' | Out-File 'out.txt' -Append

# Encoding
Get-Content 'file.txt' -Encoding UTF8
Set-Content 'out.txt' 'text' -Encoding UTF8

CSV Operations

# Import CSV
$users = Import-Csv 'users.csv'
$users | ForEach-Object {
    Write-Host "$($_.Name): $($_.Email)"
}

# Export CSV
Get-Process | Select-Object Name, Id, CPU |
    Export-Csv 'processes.csv' -NoTypeInformation

# Create CSV from objects
$data = @(
    [PSCustomObject]@{Name='Alice'; Age=30; City='NY'}
    [PSCustomObject]@{Name='Bob';   Age=25; City='LA'}
)
$data | Export-Csv 'people.csv' -NoTypeInformation

JSON Operations

# Read JSON
$json = Get-Content 'config.json' -Raw | ConvertFrom-Json
Write-Host $json.setting1

# Write JSON
$config = @{
    ServerUrl = 'https://api.example.com'
    Timeout   = 30
    Debug     = $false
}
$config | ConvertTo-Json | Set-Content 'config.json'

# Pretty print
$config | ConvertTo-Json -Depth 5

# Nested JSON
$nested = '{ "name": "Alice", "address": { "city": "NY" } }' | ConvertFrom-Json
$nested.address.city   # NY

XML Operations

# Load XML
[xml]$doc = Get-Content 'data.xml'
$doc.root.item

# XPath query
Select-Xml -Path 'data.xml' -XPath '//user[@active="true"]'

# Create XML
$xml = [xml]'<root><item>value</item></root>'
$newNode = $xml.CreateElement('item2')
$newNode.InnerText = 'value2'
$xml.root.AppendChild($newNode)
$xml.Save('output.xml')
