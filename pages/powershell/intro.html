1. Introduction to PowerShell

PowerShell is a cross-platform task automation solution made up of a command-line shell, a scripting language, and a configuration management framework.

What is PowerShell?

PowerShell is built on .NET and works with objects rather than plain text, making it more powerful than traditional shells. It runs on Windows, macOS, and Linux.

Check PowerShell Version

$PSVersionTable
$PSVersionTable.PSVersion

Get Help on Any Command

Get-Help Get-Process
Get-Help Get-Process -Examples
Get-Help Get-Process -Full

Discover Commands

Get-Command
Get-Command -Verb Get
Get-Command -Noun Process
Get-Command *network*

Execution Policy

PowerShell uses execution policies to control which scripts can run. Common policies: Restricted, AllSigned, RemoteSigned, Unrestricted, Bypass.

Get and Set Execution Policy

Get-ExecutionPolicy
Get-ExecutionPolicy -List
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
Set-ExecutionPolicy Bypass -Scope Process

PowerShell ISE vs VS Code

PowerShell ISE is the built-in editor for Windows. VS Code with the PowerShell extension is the modern recommended IDE for PowerShell scripting.

Open PowerShell ISE

# From Run dialog
powershell_ise.exe

# Or from PowerShell terminal
ise
